# CI/CD Skill

Configure and optimize continuous integration and deployment pipelines.

## When to Use

- Setting up CI/CD from scratch
- Optimizing build times
- Configuring deployment workflows
- Managing secrets in CI
- Troubleshooting build failures

## Steps

1. Identify the CI/CD platform (GitHub Actions, GitLab CI, Jenkins)
2. Review existing workflow configuration
3. Implement or modify pipeline stages
4. Test the workflow
5. Document any new processes

## Example Triggers

- "set up CI for this project"
- "optimize the GitHub Actions workflow"
- "add a deployment stage to the pipeline"
